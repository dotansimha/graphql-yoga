# Integration with SvelteKit

[SvelteKit is the fastest way to build svelte apps.](https://kit.svelte.dev/) It is very simple and lets you build the frontend & backend in a single place.

You can add GraphQL Yoga with a few lines of code and get the benefits of GraphQL & SvelteKit at the same time. [Envelop ecosystem](https://www.envelop.dev) for example!

## Installation

In a SvelteKit project:

<PackageCmd packages={['graphql', '@graphql-yoga/common']} />

## Example

### Create your GraphQL Endpoint

Create the file `src/routes/api/graphql.ts`:

```ts filename="src/routes/api/graphql.ts"
import { createServer } from '@graphql-yoga/common'
import type { RequestEvent } from '@sveltejs/kit'

const yogaApp = createServer<RequestEvent>({
  schema: {
    typeDefs: `
			type Query {
				hello: String
			}
		`,
    resolvers: {
      Query: {
        hello: () => 'SvelteKit - GraphQL Yoga',
      },
    },
  },
  graphiql: {
    endpoint: '/api/graphql',
  },
})

export { yogaApp as get, yogaApp as post }
```
