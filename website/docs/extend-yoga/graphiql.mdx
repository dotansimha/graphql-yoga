[GraphiQL is an in-browser IDE for writing, validating, and testing GraphQL queries.](https://github.com/graphql/graphiql)

You can configure or completely disable GraphiQL with the `graphiql` option. By default, GraphiQL is only enabled in development mode.

## Example

```ts
import { createGraphQLServer } from 'graphql-yoga'

// Provide your schema
const server = createGraphQLServer({
  typeDefs: /* GraphQL */ `
    type Query {
      greetings: String!
    }
  `,
  resolvers: {
    Query: {
      greetings: () => 'Hello World!',
    },
  },
  graphiql: {
    // See all options here: https://github.com/graphql/graphiql/tree/main/packages/graphiql
    defaultQuery: /* GraphQL */ `
      query {
        greetings
      }
    `,
  },
})
// Start the server and explore http://localhost:4000/graphql
server.start()
```
